# My Arcade Forest

## Basket Blast

### Objectif principal
- Le but ici est d'atteindre le plus haut score en gérant intelligemment les objets qui tombent dans les paniers.

### Les différents objets
- **Les fruits** : Ce sont les objets qui doivent tomber dans vos paniers. Ne les récupérez pas, sinon vous perdrez des points.
- **Les insectes / Poubelles** : Ce sont les objets que vous devez récupérer. S'ils tombent dans les paniers, vous perdrez des points de vie.

### Déplacement
- Swipez de gauche à droite. Il y a 3 endroits où le joueur peut se déplacer : le milieu et les côtés, où se trouvent les paniers avec les objets qui tombent.

### Points de vie
- Le joueur commence avec 3 points de vie. S'il les perd tous, c'est GAME OVER.

### Score
- Pour augmenter votre score, il faut faire rentrer le plus de fruits possible dans les paniers.
- Si vous récupérez un fruit et l'empêchez de tomber dans le panier, c'est -3 points.
- Le joueur pourra voir son high score en jeu.

### Augmentation de la difficulté
- Au fil du temps, le joueur atteindra plusieurs paliers de score. Ces paliers correspondront à des augmentations de difficulté qui accéléreront le rythme du jeu.
- Augmentation de la vitesse à laquelle les objets tombent, augmentation du rythme auquel les objets apparaissent.

### Menus
- Menu Pause
- Menu Game Over 

### Ce que j'ai essayé de faire 
- Faire en sorte que le personnage récupère l'objet afin de devoir retourner au milieu pour s'en débarrasser et de n'en porter qu'un seul.

## Beat Chopper

### Objectif principal
- Le but est de couper le plus de bois possible avant la fin de la musique

### Les types d'arbres
-**Les pins** : Il faut simplement toucher ces arbres pour les couper
-**Les chênes** : Il faut charger avant de pouvoir couper cet arbre en glissant vers le bas. Un effet lumineux permet au joueur de savoir si il est entrain de charger.

### Déplacement
- Glissez de gauche à droite pour déplacer le joueur, il suit votre doigt.

### Score
- Récupérez le plus d'abres pour obtenir le 5 étoiles maximum.

### Points de vie
- La vie est représenté par une barre qui baisse en fonction des arbres qui n'ont pas étés récupéré.
- Si la barre arrive à Zéro, c'est Game Over

### Menus
- Menu Pause
- Menu Game Over
- Menu Victoire

### Ce que j'ai essayé de faire 
- Faire en sorte que le joueur ait une animation lorsqu'il touche un arbre.

## Forest Glide

### Objectif principal
- Récupérer le plus possible de ballons en survivant aux obstacles et en parvenant à la fin du niveau.

### Les objets présents
-**Les Ballons** : Les ballons augmentent votre score, il en existe 3 sorte (1, 2 & 3 Ballons)
-**Les spike** : Peuvent être immobiles ou mobiles, enlève un point de vie si on les touche.

### Déplacement
- Glissez votre doigt pour donner une impulsion au joueur dans la direction voulue.

### Score
- Récupérez le plus de ballons possibles pour obtenir le meilleur high score.

### Points de vie
- Le joueur commence avec 3 points de vie. S'il les perd tous, c'est GAME OVER.
- Si le joueur descend trop bas dans la forêt, c'est GAME OVER.

### Augmentation de la difficulté
- Au fur et à mesure du jeu, tout les 5 points, la vitesse de la caméra augmente ce qui incite le joueur à faire plus de mouvements.

### Condition de victoire
- Se poser sur le podium qui se trouve à la fin du niveau.

### Menus
- Menu Pause
- Menu Game Over 
- Menu Victoire
